# v0.6.1 小需求与体验优化记录

这个文件用于记录 v0.6.1 周期内较小的需求、体验优化和界面修正。后续同类变更继续追加到这里，避免为每个小项单独创建发布记录文件。

## 2026-05-16

### 关闭主窗口时退出应用

- 修复点击主窗口关闭按钮后应用只隐藏到后台的问题（PR #68）。
- 旧行为会让窗口消失但进程继续运行；在没有系统托盘入口的情况下，用户无法从 Dock 或任务栏重新打开主窗口。
- 新行为改为关闭主窗口时直接退出应用，用户再次点击应用图标会重新启动应用。
- 该修复适用于 macOS 和 Windows 的窗口关闭行为。
- 修复验证：`npm run check`。
